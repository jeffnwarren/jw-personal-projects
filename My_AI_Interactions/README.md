# My AI Interactions

Notes and patterns from real AI-assisted development sessions — what worked, what to repeat, and why.

## Contents

- **[effective-ai-interaction-patterns.md](effective-ai-interaction-patterns.md)** — The main document. 18 patterns organized by session phase (starting, framing requests, watching for issues, stepping back, wrapping up), with real examples and a quick-reference table.

## History

This document was consolidated from two earlier files:

- `how-to-work-well-with-ai.md` — 10 patterns from an extended gamebook QA testing session, written in a narrative/reflective style.
- `my_ai_interactions_notes.md` — 18 patterns accumulated across multiple sessions (EDI processing, git workflows, test automation), using a "What I did / Why it worked / The pattern" structure.

The two files had significant overlap — 8 topic clusters were duplicated across them. They were merged into a single file that:

- Combined duplicate patterns, keeping the strongest examples from each source
- Organized patterns by phase of interaction (start → frame → watch → step back → wrap up) instead of chronological order
- Unified the writing style
- Preserved all unique patterns from both files

The original files have been removed to avoid confusion.
